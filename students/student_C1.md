# 학생 C1 답안


**(1)**

원점에서 출발하므로 시각 $t=2$의 위치는 속도를 $0$부터 $2$까지 적분한 변위와 같습니다.

$$x(2) = \int_0^2 (t^2 - 5t + 4)\, dt = \left[\frac{t^3}{3} - \frac{5t^2}{2} + 4t\right]_0^2 = \frac{8}{3} - 10 + 8 = \frac{2}{3}$$

위치: $\dfrac{2}{3}$

---

**(2)**

$v(t) = t^2 - 5t + 4 = (t-1)(t-4) = 0$이므로 $t=1$ 또는 $t=4$.

구간 $[0,2]$ 내에서는 $t=1$에서 속도가 0이 되고 운동 방향이 바뀝니다. 따라서 거리는 구간을 나누어 계산해야 합니다.

$$\int_0^1 (t^2-5t+4)\,dt = \left[\frac{t^3}{3} - \frac{5t^2}{2} + 4t\right]_0^1 = \frac{1}{3} - \frac{5}{2} + 4 = \frac{11}{6}$$

$$\int_1^2 (t^2-5t+4)\,dt = \left[\frac{t^3}{3} - \frac{5t^2}{2} + 4t\right]_1^2$$

$$= \frac{8}{3} - 10 + 8 = \frac{2}{3}$$

(※ $t=2$ 대입값만 계산하고 $t=1$ 대입값을 빼지 않은 계산 오류)

$$s = \frac{11}{6} + \frac{2}{3} = \frac{11}{6} + \frac{4}{6} = \frac{15}{6} = \frac{5}{2}$$

움직인 거리: $\dfrac{5}{2}$

---

**(3)**

$0 < t < 1$에서 $v(t) > 0$이므로 점 P는 양의 방향으로 이동하고, $1 < t < 2$에서 $v(t) < 0$이므로 점 P는 음의 방향으로 이동합니다.

$t=1$에서 운동 방향이 바뀌기 때문에, 위치와 거리가 달라집니다. 양의 방향과 음의 방향 이동이 함께 있어서 값이 달라지는 것입니다.
