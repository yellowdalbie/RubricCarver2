# 학생 A2 답안


**(1)**

점 P는 시각 $t=0$에 원점에서 출발하므로, 시각 $t=2$에서의 위치는 속도를 시간에 대해 적분한 값(변위)과 같습니다.

$$x(2) = x(0) + \int_0^2 v(t)\, dt = 0 + \int_0^2 (t^2 - 5t + 4)\, dt$$

$$= \left[\frac{t^3}{3} - \frac{5t^2}{2} + 4t\right]_0^2 = \left(\frac{8}{3} - 10 + 8\right) - 0 = \frac{8}{3} - 2 = \frac{2}{3}$$

$t=2$일 때 점 P의 위치: $\dfrac{2}{3}$

---

**(2)**

$v(t) = t^2 - 5t + 4 = (t-1)(t-4)$이므로, 구간 $[0,2]$ 안에서 $v(t) = 0$이 되는 시각은 $t=1$입니다.

$v(t)$는 아래로 볼록한 이차함수로, $t=1$ 이전에는 양수, $t=1$ 이후(구간 내)에는 음수입니다. 따라서 $t=1$에서 점 P의 운동 방향이 전환됩니다. 움직인 거리는 $v(t)$ 그래프와 $t$축 사이의 넓이이므로:

$$s = \int_0^2 |v(t)|\, dt = \int_0^1 v(t)\, dt + \int_1^2 (-v(t))\, dt$$

$$\int_0^1 (t^2-5t+4)\,dt = \left[\frac{t^3}{3} - \frac{5t^2}{2} + 4t\right]_0^1 = \frac{1}{3} - \frac{5}{2} + 4 = \frac{11}{6}$$

$$\int_1^2 (t^2-5t+4)\,dt = \frac{2}{3} - \frac{11}{6} = -\frac{7}{6}$$
$$\therefore \int_1^2 |v(t)|\,dt = \frac{7}{6}$$

$$s = \frac{11}{6} + \frac{7}{6} = \frac{18}{6} = 3$$

움직인 거리: $\mathbf{3}$

---

**(3)**

(1)의 위치 $\dfrac{2}{3}$과 (2)의 거리 $3$이 다른 이유는 $v(t)$의 부호가 중간에 바뀌기 때문입니다.

$0 < t < 1$에서 $v(t) > 0$이므로 점 P는 양의 방향으로 이동하고, $1 < t < 2$에서 $v(t) < 0$이므로 점 P는 음의 방향으로 이동합니다.

**위치**를 구할 때는 $\displaystyle\int_0^2 v(t)\,dt$를 사용합니다. 이 식은 $t$축 위쪽 넓이에서 아래쪽 넓이를 빼는 것이므로, 양의 방향 이동분($\frac{11}{6}$)과 음의 방향 이동분($\frac{7}{6}$)이 서로 상쇄되어 결과가 $\frac{2}{3}$으로 작아집니다.

**거리**를 구할 때는 $\displaystyle\int_0^2 |v(t)|\,dt$를 사용합니다. 이 식은 $t$축 위쪽과 아래쪽 넓이를 모두 더하는 것이므로, 음의 방향 이동분도 양수로 취급하여 누적되고 결과가 $3$으로 커집니다.

결론적으로, 속도 부호의 전환($t=1$ 기준, 양→음)으로 인해 단순 정적분(위치)은 상쇄 효과로 작아지고, 절댓값 정적분(거리)은 모든 이동을 누적하여 커지기 때문에 두 값이 다르게 나타납니다.
