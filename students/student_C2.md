# 학생 C2 답안


**(1)**

$$x(2) = \int_0^2 (t^2-5t+4)\,dt = \left[\frac{t^3}{3} - \frac{5t^2}{2} + 4t\right]_0^2 = \frac{8}{3} - 10 + 8 = \frac{2}{3}$$

원점 출발이므로 위치: $\dfrac{2}{3}$

---

**(2)**

$v(t) = (t-1)(t-4) = 0 \Rightarrow t=1$ 또는 $t=4$. 구간 $[0,2]$에서 영점은 $t=1$.

$t=1$에서 속도의 부호가 양에서 음으로 바뀌므로 운동 방향이 전환됩니다.

$$s = \int_0^1(t^2-5t+4)\,dt - \int_1^2(t^2-5t+4)\,dt = \frac{11}{6} - \left(-\frac{7}{6}\right) = 3$$

움직인 거리: $\mathbf{3}$

---

**(3)**

$0 < t < 1$에서 $v(t) > 0$이고, $1 < t < 2$에서 $v(t) < 0$입니다.

속도의 부호가 $t=1$에서 바뀌기 때문에 위치와 거리가 다르게 나타납니다.
