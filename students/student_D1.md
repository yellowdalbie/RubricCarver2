# 학생 D1 답안


**(1)**

속도를 적분하면 위치가 됩니다. 원점에서 출발하므로:

$$x(2) = \int_0^2 (t^2 - 5t + 4)\, dt$$

부정적분: $\int (t^2 - 5t + 4)\,dt = \frac{t^3}{3} - 5t^2 + 4t$

$$x(2) = \left[\frac{t^3}{3} - 5t^2 + 4t\right]_0^2 = \frac{8}{3} - 20 + 8 = \frac{8}{3} - 12 = \frac{8 - 36}{3} = -\frac{28}{3}$$

위치: $-\dfrac{28}{3}$
