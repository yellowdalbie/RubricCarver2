# 학생 B1 답안


**(1)**

위치 = 속도의 정적분(변위). 원점 출발이므로:

$$x(2) = \int_0^2 (t^2 - 5t + 4)\, dt = \left[\frac{t^3}{3} - \frac{5t^2}{2} + 4t\right]_0^2 = \frac{8}{3} - 10 + 8 = \frac{2}{3}$$

위치: $\dfrac{2}{3}$

---

**(2)**

$v(t) = (t-1)(t-4) = 0$이므로 $t=1$ 또는 $t=4$. 구간 $[0,2]$ 내에서는 $t=1$에서 속도가 0이 됩니다. $t=1$ 전후로 속도의 부호가 바뀌므로 운동 방향이 바뀝니다.

$$s = \int_0^1 (t^2-5t+4)\,dt - \int_1^2 (t^2-5t+4)\,dt = \frac{11}{6} - \left(-\frac{7}{6}\right) = \frac{11}{6} + \frac{7}{6} = 3$$

움직인 거리: $\mathbf{3}$

---

**(3)**

$0 < t < 1$에서 $v(t) > 0$이므로 점 P는 양의 방향으로 이동합니다.
$1 < t < 2$에서 $v(t) < 0$이므로 점 P는 음의 방향으로 이동합니다.

$t=1$에서 운동 방향이 바뀌기 때문에, 양의 방향 이동량과 음의 방향 이동량이 상쇄되어 위치의 최종값이 작아집니다. 반면 거리는 방향에 상관없이 이동량을 모두 더하므로 위치보다 더 크게 나옵니다. 그래서 두 값이 다릅니다.
