# 학생 A1 답안


**(1)**

원점(위치 0)에서 출발하므로, 시각 $t=2$에서의 위치 $x(2)$는 속도 함수를 $t=0$부터 $t=2$까지 정적분한 값(변위)과 같습니다.

$$x(2) = \int_0^2 (t^2 - 5t + 4)\, dt = \left[\frac{t^3}{3} - \frac{5t^2}{2} + 4t\right]_0^2$$

$$= \frac{8}{3} - \frac{5 \cdot 4}{2} + 4 \cdot 2 - 0 = \frac{8}{3} - 10 + 8 = \frac{8}{3} - 2 = \frac{2}{3}$$

따라서 시각 $t=2$일 때 점 P의 위치는 $\dfrac{2}{3}$입니다.

---

**(2)**

움직인 거리를 구하려면 속도의 부호가 바뀌는 시점을 먼저 찾아야 합니다.

$$v(t) = t^2 - 5t + 4 = (t-1)(t-4) = 0 \implies t = 1 \text{ 또는 } t = 4$$

구간 $[0, 2]$에서 영점은 $t=1$입니다. $v(0) = 4 > 0$이고 $v(1.5) = 2.25 - 7.5 + 4 = -1.25 < 0$이므로, $t=1$에서 $v(t)$의 부호가 양(+)에서 음(-)으로 바뀝니다. 즉 점 P는 $t=1$에서 운동 방향을 바꿉니다.

따라서 움직인 거리는:

$$s = \int_0^2 |v(t)|\, dt = \int_0^1 v(t)\, dt - \int_1^2 v(t)\, dt$$

$$\int_0^1 (t^2 - 5t + 4)\, dt = \left[\frac{t^3}{3} - \frac{5t^2}{2} + 4t\right]_0^1 = \frac{1}{3} - \frac{5}{2} + 4 = \frac{2}{6} - \frac{15}{6} + \frac{24}{6} = \frac{11}{6}$$

$$\int_1^2 (t^2 - 5t + 4)\, dt = \left[\frac{t^3}{3} - \frac{5t^2}{2} + 4t\right]_1^2 = \frac{2}{3} - \frac{11}{6} = \frac{4}{6} - \frac{11}{6} = -\frac{7}{6}$$

$$s = \frac{11}{6} - \left(-\frac{7}{6}\right) = \frac{11}{6} + \frac{7}{6} = \frac{18}{6} = 3$$

시각 $t=0$에서 $t=2$까지 점 P가 움직인 거리는 $\mathbf{3}$입니다.

---

**(3)**

(1)에서 구한 위치는 $\dfrac{2}{3}$이고, (2)에서 구한 움직인 거리는 $3$으로 서로 다릅니다. 이 차이는 속도 함수 $v(t)$의 부호 변화에서 비롯됩니다.

- $0 < t < 1$ 구간에서는 $v(t) > 0$이므로, 점 P는 양의 방향(수직선의 오른쪽)으로 이동합니다.
- $1 < t < 2$ 구간에서는 $v(t) < 0$이므로, 점 P는 음의 방향(수직선의 왼쪽)으로 이동합니다.

**위치(변위)**는 단순 정적분 $\int_0^2 v(t)\, dt$를 계산한 것이므로, 양의 방향 이동량과 음의 방향 이동량이 서로 상쇄됩니다. 그 결과 실제로 얼마나 왔다 갔다 했는지와 무관하게 최종 도착 지점만 반영되어 값이 작아집니다.

**움직인 거리**는 $\int_0^2 |v(t)|\, dt$를 계산한 것이므로, 음의 방향으로 이동한 양도 절댓값을 취해 양수로 더해집니다. 따라서 두 방향의 이동량이 모두 누적되어 위치보다 큰 값이 나옵니다.

요약하면, 속도의 부호가 $t=1$에서 양(+)에서 음(-)으로 바뀌기 때문에 단순 적분(위치)과 절댓값 적분(거리)의 계산 방식이 달라지고, 이로 인해 두 값이 서로 다르게 나타납니다.
