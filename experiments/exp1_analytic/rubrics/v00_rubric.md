| **C1-1** | 시각 $t=2$에서의 위치를 구하기 위해 정적분 식 $\int_0^2 (t^2-5t+4)\,dt$ 를 수식으로 명시했는가? | [절차] |
| **C1-2** | 원점에서 출발한 점의 위치가 속도 함수를 적분한 값(변위)과 같다는 관계를 수식 또는 서술로 표현했는가? | [개념] |
| **C2-1** | $v(t) = t^2 - 5t + 4 = 0$을 풀어 구간 $[0, 2]$ 내 영점 $t = 1$을 도출했는가? | [절차] |
| **C2-2** | $t = 1$에서 속도의 부호가 양(+)에서 음(−)으로 바뀌므로 점 P의 운동 방향이 전환된다는 사실을 서술했는가? | [개념] |
| **C2-3** | 구간 $[0,1]$과 $[1,2]$로 나누어 절댓값 적분하고 최종 움직인 거리 $3$을 도출했는가? | [절차] |
| **C3-1** | $0 < t < 1$ 구간에서 $v(t) > 0$이므로 점 P가 양의 방향으로 이동함을 명시했는가? | [개념] |
| **C3-2** | $1 < t < 2$ 구간에서 $v(t) < 0$이므로 점 P가 음의 방향으로 이동함을 명시했는가? | [개념] |
| **C3-3** | 단순 정적분(변위·위치)과 절댓값 정적분(거리)이 서로 다른 계산임을 수식 또는 표현으로 구분했는가? | [절차] |
| **C3-4** | 위치(변위)는 양·음 방향 이동량이 상쇄되고, 거리는 음의 방향 이동도 양수로 누적된다는 메커니즘을 서술했는가? | [개념] |
| **C3-5** | 속도 부호 변화 → 방향 전환 → 상쇄(위치) vs. 누적(거리)이라는 인과 논리를 완결된 서술로 제시했는가? | [개념] |
