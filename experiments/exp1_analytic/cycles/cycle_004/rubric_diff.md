# 루브릭 기준 변경사항

## 변경된 기준 (5개)

### C1-1
**이전:** 시각 $t=2$에서의 위치를 구하기 위해 정적분 식 $\int_0^2 (t^2-5t+4)\,dt$ 를 수식으로 명시했는가?
**이후:** 위치를 구하기 위해 속도 함수를 주어진 구간($[0,2]$)에서 정적분(또는 '누적 계산')해야 함을 수식 또는 그에 상응하는 서술로 표현했는가?

### C2-2
**이전:** $t = 1$에서 속도의 부호가 양(+)에서 음(−)으로 바뀌므로 점 P의 운동 방향이 전환된다는 사실을 서술했는가?
**이후:** $t=1$에서 운동 방향이 바뀌거나 속도의 부호가 바뀐다는 핵심 사실을 서술했는가?

### C2-3
**이전:** 구간을 나누어 각 구간의 이동량을 계산한 후, 이들의 합으로 총 움직인 거리 3을 구했는가? (수식 또는 서술 모두 인정)
**이후:** 구간을 나누어 각 구간 이동량의 **절댓값(또는 양수 값)을 합산**하여 총거리를 구하는 과정을 제시했는가? (개별 계산의 정오 여부와 무관)

### C3-1
**이전:** $0 < t < 1$ 구간에서 $v(t) > 0$이므로 점 P가 양의 방향으로 이동함을 명시했는가?
**이후:** $0 < t < 1$ 구간에서 속도의 부호가 양(+)이며, 이로 인해 점 P가 양의 방향으로 이동한다는 **관계**를 서술했는가?

### C3-2
**이전:** $1 < t < 2$ 구간에서 $v(t) < 0$이므로 점 P가 음의 방향으로 이동함을 명시했는가?
**이후:** $1 < t < 2$ 구간에서 속도의 부호가 음(-)이며, 이로 인해 점 P가 음의 방향으로 이동한다는 **관계**를 서술했는가?
