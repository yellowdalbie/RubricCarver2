- **성취기준**: [12미적Ⅰ-03-06] 적분을 속도와 거리에 대한 문제에 활용하고, 그 유용성을 인식할 수 있다.

## 채점 기준표

| 코드 | 이진 채점 기준 | 유형 |
|------|---------------|------|
| **C1-1** | 위치를 구하는 **과정**으로, 정적분 식($\int_0^2 v(t)dt$)을 명시하거나 그에 준하는 논리적 절차를 제시했는가? **(단순히 최종 값만 제시한 경우는 불인정)** | [절차] |
| **C1-2** | 원점에서 출발한 점의 위치가 속도 함수를 적분한 값(변위)과 같다는 관계를 수식 또는 서술로 표현했는가? | [개념] |
| **C2-1** | v(t)=0의 해가 t=1임을 방정식을 풀거나 값을 대입하여 확인하는 등, 논리적인 방법으로 보였는가? | [절차] |
| **C2-2** | $t=1$에서 운동 방향이 바뀌거나 속도의 부호가 바뀐다는 핵심 사실을 서술했는가? | [개념] |
| **C2-3** | 음의 방향 이동량에 **절댓값을 취하거나 부호를 바꾸어** 양수로 만든 뒤, 양의 방향 이동량과 합산하는 과정을 수식($\int_0^1 v(t)dt - \int_1^2 v(t)dt$ 또는 $\int_0^2 |v(t)|dt$ 등)이나 서술로 명확히 제시했는가? | [절차] |
| **C3-1** | $0 < t < 1$ 구간에서 속도의 부호가 양(+)이며, 이로 인해 점 P가 양의 방향으로 이동한다는 **관계**를 서술했는가? | [개념] |
| **C3-2** | $1 < t < 2$ 구간에서 속도의 부호가 음(-)이며, 이로 인해 점 P가 음의 방향으로 이동한다는 **관계**를 서술했는가? | [개념] |
| **C3-3** | 위치는 이동량을 **부호 그대로 합산(상쇄)**하고, 거리는 **절댓값을 합산(누적)**한다는 계산 방식의 차이를 명시적으로 서술했는가? | [절차] |
| **C3-4** | 위치와 거리의 차이가 발생하는 이유로, 위치는 양/음의 이동량이 서로 **'상쇄'**되는 반면, 거리는 모든 이동량이 **'누적'**(또는 '합산')된다는 핵심 메커니즘을 **명시적으로** 서술했는가? | [개념] |
| **C3-5** | 속도 부호 변화 → 방향 전환 → 상쇄(위치) vs. 누적(거리)이라는 인과 논리를 완결된 서술로 제시했는가? | [개념] |
