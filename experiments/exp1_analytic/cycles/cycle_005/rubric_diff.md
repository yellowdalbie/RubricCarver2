# 루브릭 기준 변경사항

## 변경된 기준 (5개)

### C1-1
**이전:** 풀이 과정 중에 시각 $t=2$에서의 위치를 구하기 위해 정적분 식 $\int_0^2 (t^2-5t+4)\,dt$ 를 수식으로 명시했는가?
**이후:** 위치를 구하는 **과정**으로, 속도 함수를 주어진 구간($[0,2]$)에서 정적분하는 **계산식이나 그에 준하는 논리적 절차**를 제시했는가? (단순히 최종 값만 제시한 경우는 불인정)

### C2-3
**이전:** 구간을 나누어 각 구간의 이동량을 계산한 후, 이들의 합으로 총 움직인 거리를 구하는 과정을 제시했는가? (최종 계산 결과의 정오 여부와 무관)
**이후:** 구간을 나누어 각 구간 이동량의 **절댓값(또는 양수 값)을 합산**하여 총거리를 구하는 과정을 제시했는가? (개별 계산의 정오 여부와 무관)

### C3-1
**이전:** $0 < t < 1$ 구간에서 $v(t) > 0$임을 근거로 점 P가 양의 방향으로 이동함을 명시했는가?
**이후:** $0 < t < 1$ 구간에서 속도의 부호가 양(+)이며, 이로 인해 점 P가 양의 방향으로 이동한다는 **관계**를 서술했는가?

### C3-2
**이전:** $1 < t < 2$ 구간에서 $v(t) < 0$임을 근거로 점 P가 음의 방향으로 이동함을 명시했는가?
**이후:** $1 < t < 2$ 구간에서 속도의 부호가 음(-)이며, 이로 인해 점 P가 음의 방향으로 이동한다는 **관계**를 서술했는가?

### C3-4
**이전:** 위치는 양/음의 이동량이 서로 '상쇄'되고, 거리는 모든 이동량이 합산(또는 '누적')된다는 핵심 메커니즘을 서술했는가?
**이후:** 위치와 거리의 차이가 발생하는 이유로, 위치는 양/음의 이동량이 서로 **'상쇄'**되는 반면, 거리는 모든 이동량이 **'누적'**(또는 '합산')된다는 핵심 메커니즘을 **명시적으로** 서술했는가?

## 변경 없는 기준 (5개)
C1-2, C2-1, C2-2, C3-3, C3-5
