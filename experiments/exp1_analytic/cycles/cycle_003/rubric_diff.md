**이후:** 풀이 과정 중에 시각 $t=2$에서의 위치를 구하기 위해 정적분 식 $\int_0^2 (t^2-5t+4)\,dt$ 를 수식으로 명시했는가?

### C2-1
**이전:** v(t)=0 방정식을 세우고, 이를 풀어 영점 t=1을 구하는 과정을 제시했는가?
**이후:** v(t)=0의 해가 t=1임을 방정식을 풀거나 값을 대입하여 확인하는 등, 논리적인 방법으로 보였는가?

### C2-2
**이전:** $t = 1$에서 속도의 부호가 양(+)에서 음(−)으로 바뀌므로 점 P의 운동 방향이 전환된다는 사실을 서술했는가?
**이후:** $t=1$에서 운동 방향이 바뀌거나 속도의 부호가 바뀐다는 핵심 사실을 서술했는가?

### C2-3
**이전:** 구간을 나누어 각 구간의 이동량을 계산한 후, 이들의 합으로 총 움직인 거리 3을 구했는가? (수식 또는 서술 모두 인정)
**이후:** 구간을 나누어 각 구간의 이동량을 계산한 후, 이들의 합으로 총 움직인 거리를 구하는 과정을 제시했는가? (최종 계산 결과의 정오 여부와 무관)

### C3-1
**이전:** $0 < t < 1$ 구간에서 $v(t) > 0$이므로 점 P가 양의 방향으로 이동함을 명시했는가?
**이후:** $0 < t < 1$ 구간에서 $v(t) > 0$임을 근거로 점 P가 양의 방향으로 이동함을 명시했는가?

### C3-2
**이전:** $1 < t < 2$ 구간에서 $v(t) < 0$이므로 점 P가 음의 방향으로 이동함을 명시했는가?
**이후:** $1 < t < 2$ 구간에서 $v(t) < 0$임을 근거로 점 P가 음의 방향으로 이동함을 명시했는가?

### C3-4
**이전:** 위치는 양/음의 이동량이 서로 '상쇄'되고, 거리는 모든 이동량이 '누적'된다는 핵심 메커니즘을 명시적으로 서술했는가?
**이후:** 위치는 양/음의 이동량이 서로 '상쇄'되고, 거리는 모든 이동량이 합산(또는 '누적')된다는 핵심 메커니즘을 서술했는가?
