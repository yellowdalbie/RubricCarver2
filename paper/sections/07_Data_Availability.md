# 7. 데이터 공개

본 연구는 평가 과정의 투명성과 재현성을 보장하기 위해 다중 에이전트가 생성한 모든 원시 데이터와 소스 코드를 외부에 전면 공개한다. 지면의 한계상 본문에 수록하기 어려운 방대한 양의 에이전트 통신 로그, 학생 답안별 채점 결과, 그리고 사이클별 루브릭 개정 이력은 연구 대시보드(<https://yellowdalbie.github.io/RubricCarver2/>)를 통해 상호작용 가능한 형태로 제공된다. 독자는 해당 웹 대시보드에 접속하여 에이전트별 시스템 프롬프트 원문을 열람하고, 특정 실험 사이클에서 발생한 팩트체크 및 심의 과정을 직접 추적할 수 있다. 

또한, 웹 서버 만료로 인한 자료 유실을 방지하고 장기적인 접근성을 확보하기 위해, 본 연구에 사용된 시스템의 전체 소스 코드와 원시 데이터 파일은 깃허브 공개 저장소(<https://github.com/yellowdalbie/RubricCarver2>)에 영구적으로 보존 및 배포된다.
