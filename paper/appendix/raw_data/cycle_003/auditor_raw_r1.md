## 3. 반례 검토 (STEP 3)
- 반례 존재 여부: 없음

- 절차/개념 비율 확인: [절차] 4개 / [개념] 6개

- **반례 검토 상세**:
  "이 기준대로 0점이 되지만, 해당 수학 개념을 이해하는 학생의 답안"을 상상하기 어렵습니다.
  개념을 이해한 학생은 최소한 다음 중 하나는 수행할 것입니다.
  1.  속도가 0이 되는 지점($t=1$)을 찾는다. → **C2-1** 득점.
  2.  그래프를 그려 $t=1$에서 x축과 만남을 보인다. → **C2-1** 득점.
  3.  구간을 나누어 계산하는 과정을 보인다. → **C2-3** 득점.
  4.  '방향이 바뀐다' 또는 '속도 부호가 바뀐다'고 서술한다. → **C2-2** 득점.
  
  어떤 형태의 이해를 표현하더라도 최소 1점 이상을 획득하게 설계되어 있어, 개념을 이해하고도 0점을 받는 '거짓 음성(false negative)' 오류의 가능성은 거의 없습니다. 또한, [개념] 기준의 비율이 60%로 유지되어 출제 의도인 개념적 이해 평가에 충실합니다.
